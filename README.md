# Rest Compiler

**Rest** — это компилируемый строго типизированный язык системного программирования. Он компилируется напрямую в машинный код через LLVM, обеспечивая производительность уровня C++ и предсказуемое управление памятью без сборщика мусора.

```text
Lexer → Parser → Sema → HIR → LLVM IR → Native
```

---

## Ключевые Особенности

### 1. Управление Памятью (ARC)
Мы отказались от традиционных сборщиков мусора (GC), чтобы избежать непредсказуемых пауз в работе программ.
* Память управляется автоматически через механизм подсчета ссылок (Automatic Reference Counting).
* Структуры данных и строки имеют заголовок в куче (heap) со счетчиком.
* Компилятор неявно расставляет вызовы `__rest_retain` и `__rest_release`.
* Когда счетчик ссылок падает до нуля, объект и его дочерние элементы удаляются из памяти (каскадная очистка).

### 2. Прямой доступ к железу (FFI & Указатели)
Rest дает возможность писать низкоуровневый системный код:
* **`extern fn`**: Вызов любых системных функций (например, из libc).
* **Сырые указатели**: Поддержка `*T`, `*u8`, операторов `&` (взятие адреса) и `*` (разыменование).
* **Приведение типов**: Оператор `as` и функция `sizeof` на этапе компиляции.

## Архитектура Компилятора

Компилятор Rest написан на Rust и работает по классической схеме:
1. **Лексический и Синтаксический анализ**: Парсер методом рекурсивного спуска, строящий Абстрактное Синтаксическое Дерево (AST).
2. **Семантический анализ (Sema)**: Строгая проверка типов и валидация кода.
3. **Генерация HIR**: Преобразование AST в высокоуровневое представление (High-Level IR) для упрощения кодогенерации.
4. **Кодогенерация (LLVM)**: Трансляция HIR в LLVM IR, встроенная логика ARC, JIT-компиляция и сборка в бинарный файл.

## Сборка и Запуск

Убедитесь, что у вас установлен Rust (1.85+) и LLVM 18 (`llvm-config` должен быть доступен в `PATH`).

1. Компиляция и мгновенный запуск кода (JIT):
```bash
cargo run --release --bin restc -- jit test.rest
```

2. Сборка исполняемого файла:
```bash
cargo run --release --bin restc -- build test.rest
```
