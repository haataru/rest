# Справочник по языку Rest

Язык Rest имеет C-подобный синтаксис (вдохновленный Rust и JavaScript/TypeScript), но ориентирован на простоту, автоматическое управление памятью и прямое взаимодействие с системой через FFI.

## 1. Базовые типы данных
- `int` — 64-битное знаковое целое число (`i64` в LLVM).
- `bool` — логический тип (`true` / `false`).
- `string` — строка (представляет собой указатель на структуру, управляемую ARC). Строки в Rest неизменяемы и могут конкатенироваться оператором `+`.
- `void` — отсутствие значения (используется только для возвращаемых типов функций).

## 2. Структуры данных
Структуры в Rest передаются по ссылке и автоматически управляются сборщиком мусора (ARC).

```rust
struct User {
    id: int,
    name: string,
}

fn create_user() -> User {
    let u = User {
        id: 1,
        name: "Alice"
    };
    return u; // Память освободится автоматически, когда последняя ссылка на u исчезнет
}
```

## 3. Массивы
Массивы также управляются сборщиком мусора. Доступен встроенный метод `.length()`.

```rust
let arr: int[] = [1, 2, 3];
let count = arr.length();
let first = arr[0];
```

## 4. Переменные
Переменные объявляются ключевым словом `let`. Тип переменной может быть выведен компилятором автоматически, либо указан явно.

```rust
let x = 10;
let y: int = 20;
let name = "Bob";
```

## 5. Управляющие конструкции
Поддерживаются классические `if` / `else` и цикл `while`.

```rust
if x > 10 {
    // ...
} else {
    // ...
}

let i = 0;
while i < 10 {
    i = i + 1;
}
```

## 6. Взаимодействие с системой (FFI и Указатели)

Rest предоставляет мощные средства для прямого взаимодействия с C-библиотеками и системными вызовами операционной системы (например, Linux syscalls, сокеты).

### Внешние функции (`extern fn`)
Ключевое слово `extern` используется для объявления функций, реализация которых предоставляется операционной системой или сторонними C-библиотеками. Они линкуются на этапе сборки.

```rust
extern fn socket(domain: int, type_: int, protocol: int) -> int;
extern fn close(fd: int) -> int;
extern fn read(fd: int, buf: *u8, count: int) -> int;
```

### Указатели (`*T`)
Rest поддерживает сырые указатели (Raw Pointers), которые аналогичны указателям в C. Они полезны при работе с системными API, где нужно передавать буферы памяти или структуры. На данный момент активно поддерживается универсальный байтовый указатель `*u8`.

### Оператор AddressOf (`&`)
Оператор `&` позволяет получить сырой указатель на участок памяти, в котором хранится переменная (L-Value).
```rust
let my_var = 42;
let ptr = &my_var; // ptr имеет тип *int
```

### Разыменование (`*`)
Оператор разыменования позволяет получить значение из памяти по сырому указателю.
```rust
let value = *ptr; // value == 42
```

### Приведение типов (`as`)
Оператор `as` позволяет преобразовывать типы друг в друга (кастовать). Это особенно полезно при работе с FFI:
- `int` в `*T` (`inttoptr`)
- `*T` в `int` (`ptrtoint`)
- Конвертации размеров (`trunc`, `sext`, `zext` на уровне LLVM)
```rust
let fd = 3;
let handle = fd as *u8; // Каст числа в указатель
let raw = handle as int; // Каст указателя в число
```

### Размер типа (`sizeof`)
Оператор `sizeof(T)` вычисляет размер любого типа данных на этапе компиляции (Compile-Time) в байтах, что необходимо при выделении памяти под C-структуры (`sockaddr` и др.).
```rust
let size = sizeof(int); // 8 байт
```

## 7. Функции
Функции могут принимать аргументы и возвращать значения. Если возвращаемый тип не указан, предполагается `void`.

```rust
fn add(a: int, b: int) -> int {
    return a + b;
}
```
